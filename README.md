
# Car Classification

A machine learning project that classifies car types from images using Python and deep learning techniques.

## About

This project uses a Jupyter Notebook to build and train an image classification model capable of identifying and categorizing cars based on visual features.

## Built With

- Python
- Jupyter Notebook
- scikit-learn / TensorFlow / Keras
- pandas, NumPy, Matplotlib

## Project Structure

```
car-classification/
├── src/                        # Source code
├── outputs/                    # Model outputs and results
├── car-classification-notebook.ipynb  # Main notebook
├── requirements.txt            # Dependencies
└── README.md
```

## How to Run

1. Clone the repository
   ```
   git clone https://github.com/habiba-102/car-classification.git
   cd car-classification
   ```

2. Install dependencies
   ```
   pip install -r requirements.txt
   ```

3. Open the notebook
   ```
   jupyter notebook car-classification-notebook.ipynb
   ```
